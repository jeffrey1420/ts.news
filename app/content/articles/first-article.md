# Welcome to ts.news

We're building something different here. A news site for TypeScript and web technologies that respects your time and intelligence.

## Why ts.news?

The web is flooded with content. Tutorial sites that haven't been updated in years, news aggregators that prioritize clicks over quality, and endless listicles that promise the world but deliver nothing.

We wanted something better.

**ts.news** is built on a simple premise: developers deserve well-written, accurate, and timely coverage of the technologies they use every day.

## What to Expect

We'll be covering:

- **TypeScript** — Language features, type system deep dives, best practices
- **Web Standards** — What's new in the platform, TC39 proposals, browser APIs
- **Frameworks** — Vue, React, Svelte, Nuxt, Next — whichever tools get the job done
- **Developer Experience** — Tools, workflows, and everything that makes coding better

## Built With

This site is powered by:

- **Nuxt 4** — The Vue framework we trust for building fast, modern web apps
- **Nuxt UI** — Beautiful, accessible components
- **Nuxt Content** — Markdown-based content that keeps things simple
- **TypeScript** — Because types matter

## Stay Tuned

Articles are coming. We're building in public, and the next waves will bring:

- Wave 2: Design system and polished layouts
- Wave 3: First real article research and writing
- Wave 4: SEO polish and deployment

If you're a developer who cares about quality technical content, you're in the right place.

Welcome to **ts.news**.
